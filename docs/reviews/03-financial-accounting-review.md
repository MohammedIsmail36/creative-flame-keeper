## 🔴 حرج — الضريبة غير مسجلة محاسبياً!

### المشكلة الأكبر: الضريبة تُحسب لكن لا تُسجل في القيود

- **المثال**: فاتورة بيع 1,000 + ضريبة 140 = 1,140
- **ما يحدث حالياً**:
  - مدين: العميل 1,140 ✓
  - دائن: الإيرادات 1,140 ✗ خطأ!
- **المطلوب**:
  - مدين: العميل 1,140
  - دائن: الإيرادات 1,000
  - دائن: مستحقات ضريبة المبيعات 140
- **التأثير**: الإيرادات مبالغ فيها بقيمة الضريبة! لا يمكن تتبع التزامات الضريبة
- **الملفات**: RPCs في `supabase/migrations/`, `SalesInvoiceForm.tsx`

### إضافة حسابات ضريبية مطلوبة:

```sql
-- ضريبة مبيعات مستحقة
INSERT INTO accounts (code, name, account_type) VALUES ('2102', 'مستحقات ضريبة المبيعات', 'liability');
-- ضريبة مشتريات (مدخلات)
INSERT INTO accounts (code, name, account_type) VALUES ('1105', 'ضريبة القيمة المضافة للمدخلات', 'asset');
```

## 🔴 حرج — لا قيد DB يضمن توازن القيود

### المشكلة:

- القيود تحتوي `total_debit` و `total_credit`
- لكن **لا يوجد CHECK constraint** يضمن تساويهما
- البيانات يمكن أن تتلف عبر SQL مباشر أو API

### الحل:

```sql
ALTER TABLE journal_entries ADD CONSTRAINT
  jentry_balanced CHECK (total_debit = total_credit);
```

## 🔴 حرج — تقارير مالية مفقودة

| التقرير                    | الحالة       | الأهمية                 |
| -------------------------- | ------------ | ----------------------- |
| **قائمة التدفقات النقدية** | ❌ غير موجود | إلزامي للقرارات المالية |
| **تحليل أعمار الذمم**      | ❌ غير موجود | أساسي للتحصيل           |

### تقارير موجودة وتعمل:

- ✅ ميزان المراجعة (Trial Balance)
- ✅ الميزانية العمومية (Balance Sheet) — A = L + E ✓
- ✅ قائمة الدخل (Income Statement)
- ✅ دفتر الأستاذ (Ledger)
- ✅ اليومية (Journal)
- ✅ كشف حساب عميل/مورد

## 🟡 مهم — تقييم المخزون

### الوضع الحالي:

- ✅ متوسط مرجح مُطبق (Weighted Average)
- ❌ لا يُحفظ سجل متوسط التكلفة (غير قابل للتدقيق)

---

## 🟡 مهم — إقفال السنة المالية

### ما هو مُطبق ✓:

- إقفال حسابات الإيرادات والمصروفات
- حساب صافي الدخل
- ترحيل إلى أرباح محتجزة
- إمكانية عكس القيد

### ما ينقص:

- ❌ لا يمنع الإقفال مع وجود مسودات (يحذر فقط)
- ❌ لا فحص لمطابقة AR/AP قبل الإقفال

- ✅ المخزون يُسترد
