

# خطة اختبار ذرية احترافية — النظام المحاسبي

## المنهج: عملية واحدة → تحقق شامل → تراكم

كل اختبار = عملية واحدة فقط. بعد كل عملية نتحقق من **كل مكان** يتأثر بأرقام محددة مسبقاً. الأرقام تتراكم من اختبار لآخر.

## التنفيذ التقني

1. تصفير قاعدة البيانات عبر edge function `database-backup` (reset)
2. إعادة كتابة `run-test-phases` لتنفيذ عملية واحدة بـ `phase` parameter
3. بعد كل phase: استعلام DB برمجياً للتحقق من كل نقطة
4. عند الفشل: تحديد الملف → إصلاح → إعادة من الصفر
