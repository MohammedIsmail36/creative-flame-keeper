# خطة: عرض الاسم الكامل للمنتج وإتاحة البحث في تحليل المنتجات

## الهدف
في صفحة `/reports/products` (ProductAnalytics)، عرض المنتج بالصيغة الموحّدة في النظام:
**اسم المنتج - الماركة المصنعة - رقم الموديل**
مع إتاحة البحث بأي من هذه الحقول الثلاثة.

## نطاق التغييرات
ملف واحد فقط: `src/pages/reports/ProductAnalytics.tsx`

## التفاصيل التقنية

### 1) جلب الماركة والموديل من قاعدة البيانات
في الاستعلامات الحالية يتم جلب `name, code` فقط. سنضيف `model_number` و علاقة `brand:product_brands(name)`:

- **Q1 (sales items)** السطر 254: إضافة `model_number, brand:product_brands(name)` داخل `product:products!inner(...)`.
- **Q4 (products)** السطر 306: إضافة نفس الحقلين.

### 2) توسعة الأنواع (Types)
إضافة `brand?: string | null` و `model?: string | null` إلى:
`ProductMetrics`, `TurnoverMetrics`, `ABCMetrics`, `ReturnMetrics`.

### 3) دمج الاسم الكامل عند البناء
استخدام الدالة الموحّدة الموجودة في النظام:
```ts
import { formatProductDisplay } from "@/lib/product-utils";
```
عند بناء `metrics` (السطر 380) وعند بناء `turnoverList` (~565)، `abcList` (~634)، `returnsAnalysis` (~668):
```ts
name: formatProductDisplay(p.name, p.brand?.name, p.model_number),
```
وحفظ `brand` و `model` كحقلين منفصلين أيضًا لاستخدامهما في:
- البحث (لو احتجنا حقلًا منفصلًا)
- ملفات التصدير (Excel/PDF) لإظهار أعمدة منفصلة لاحقًا (اختياري — في هذه الخطة سنُبقي عمود "المنتج" واحدًا بالاسم الكامل لكن بحيث تشمل صفوف التصدير الاسم الكامل).

### 4) البحث بالماركة والموديل
`DataTable` يستخدم Global Filter يبحث عبر القيم النصية للأعمدة الظاهرة. بما أن عمود "المنتج" سيحتوي الآن على الاسم الكامل (`الاسم - الماركة - الموديل`)، سيعمل البحث تلقائيًا على الثلاثة.

تحديثات موازية:
- تغيير الـ placeholders للبحث في كل التبويبات (سطور 2583, 2610, 2657, 2681, 2705) من:
  `"بحث بالاسم أو الكود..."` إلى `"بحث بالاسم أو الماركة أو الموديل أو الكود..."`.
- في تبويب "by-category" البحث يبقى بالتصنيف كما هو.

### 5) تحديثات العرض
- العمود `accessorKey: "name"` يبقى كما هو، لكن قيمته الآن الاسم الكامل (دون تغيير في خلايا الجدول).
- المخططات (Pie/Bar) التي تستخدم `p.name`: ستعرض الاسم الكامل. الأماكن التي تقصّ الاسم بـ `p.name.slice(0, 15)` تبقى كما هي (تحافظ على الشكل البصري).
- ملفات التصدير (Excel/PDF) ستحصل تلقائيًا على الاسم الكامل لأنها تأخذ `p.name`.

## ملخص الملفات
- `src/pages/reports/ProductAnalytics.tsx` — تعديل وحيد (queries + types + bناء metrics + placeholders).

## اختبار يدوي
1. فتح `/reports/products` — التحقق أن المنتجات تظهر بصيغة `الاسم - الماركة - الموديل`.
2. البحث بالماركة فقط → نتائج صحيحة.
3. البحث برقم الموديل → نتائج صحيحة.
4. البحث بالكود → يعمل كما كان.
5. تبديل التبويبات (الأكثر مبيعاً / الأكثر ربحية / الدوران / ABC / المرتجعات) — كلها تعرض الاسم الكامل.
6. تصدير Excel و PDF — التأكد أن عمود المنتج يحوي الاسم الكامل.
