### التفاصيل التقنية

**حساب نسب التغيير الشهرية:**

```typescript
const currMonth = new Date().getMonth();
const currMonthTotal = invoices.filter(i => new Date(i.invoice_date).getMonth() === currMonth).reduce(...);
const prevMonthTotal = invoices.filter(i => new Date(i.invoice_date).getMonth() === currMonth - 1).reduce(...);
const changePercent = prevMonthTotal > 0 ? ((currMonthTotal - prevMonthTotal) / prevMonthTotal * 100).toFixed(1) : null;
```

**السيولة النقدية:** جلب حسابات نوعها `asset` ورمزها يبدأ بـ `1101` أو `1102` (نقدية/بنوك) من `accounts` + أرصدتها من `journal_entry_lines`.

**الأصناف الأكثر مبيعاً:** تجميع `sales_invoice_items` مع join على `products` و `sales_invoices` (posted فقط).

**تصميم البطاقات:** مطابق للملف المرفق - خلفية بيضاء، حدود خفيفة، أيقونات داخل مربعات مدورة ملونة، نسب التغيير بأسهم خضراء/حمراء.
